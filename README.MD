# Jaydes4Four

A personal bio-link page for **Jaydes4Four**, based on an open-source template inspired by guns.lol and fakecrime.bio.

## How to use
Just open `index.html` in your browser (a Live Server extension works best). No setup required.

## Customization
If you want to change the **music**, **background**, or **cursor**, just replace the corresponding files in the `assets/` folder.
Keep the **same file format** (e.g. `.mp3`, `.png`, `.mp4`) to avoid issues.

You can fully edit the script, change styles, add features, etc. — go wild and make it your own.

## Add Your Own Assets

To keep this project lightweight, not every media file is guaranteed to be present. If your site looks broken or silent, add the missing files to the `assets/` folder using the exact filenames below.
